# پروژه وب‌سایت خدمات مهاجرت

این پروژه یک قالب React (Vite) آماده با Tailwind است که شامل صفحات:
- صفحه اصلی (Hero, Services, Process, About, Contact)
- بلاگ نمونه
- مقایسه کشورها
- فرم چندمرحله‌ای ساده
- بخش قیمت و نظرات

## اجرا در لوکال
1. Node.js نصب کنید.
2. در پوشه پروژه:
   ```
   npm install
   npm run dev
   ```
3. سپس به `http://localhost:5173` بروید.

## انتشار سریع (Vercel)
1. مخزن را به GitHub اضافه کنید.
2. وارد vercel.com شوید و پروژه را Import کنید.
3. تنظیمات پیش‌فرض Vite را قبول کنید و Deploy را اجرا کنید.

توجه: فایل `package.json` شامل وابستگی‌های اصلی است. اگر می‌خواهید نسخه‌ی بدون Tailwind یا بدون framer-motion داشته باشید، اطلاع دهید تا من تغییر دهم.
